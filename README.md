# Nassim API
- Backend for a stock trading app.
- See [Nassim React](https://github.com/elainechan/nassim-react) for frontend.

## Stack
- Server: Node.js, Express.js
- Database: MongoDB Atlas, Google Cloud
- Deployment: Heroku, Netlify, Travis CI
